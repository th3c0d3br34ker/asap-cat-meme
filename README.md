## ASAP Project

### Introduction

A demo project to learn the MERN stack.

### Tech Stack

- MongoDB
- Express
- React
- Node.js


### How to run

1. Clone the repo

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
npm run start
```

4. Start the client

```bash
cd client

npm run dev
```
